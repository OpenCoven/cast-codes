# Code of Conduct

CastCodes uses the Contributor Covenant Code of Conduct as its baseline.

## Expected Behavior

- Be respectful and constructive.
- Focus criticism on technical content and project outcomes.
- Assume maintainers may be handling both authorship and review.
- Avoid harassment, abuse, discriminatory language, and personal attacks.

## Enforcement

Maintainers may remove comments, close issues, block accounts, or take other moderation action when behavior damages the project or its contributors.

If a private reporting channel is not configured for this repository, ask maintainers to establish one before sharing sensitive details.
